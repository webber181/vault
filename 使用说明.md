## 七、常见问题

### Q1: Obsidian Git 插件没有自动同步
**A:** 打开 Obsidian → 设置 → 社区插件 → 确认 `Obsidian Git` 已启用 → 检查插件设置的 Pull/Push 间隔是否为 5 分钟

### Q2: 本地修改后 Git push 失败
**A:** 在终端执行：
```bash
cd ~/Library/Mobile\ Documents/iCloud\~md\~obsidian/Documents/Webber知识库
git push origin main
```
如果仍失败，可能是网络问题，稍后重试

### Q3: 服务器端没有新内容
**A:** 检查 cron 是否正常运行：
```bash
crontab -l
```
手动触发：
```bash
python3 /home/debian/.hermes/scripts/knowledge_hub.py
```

### Q4: iCloud 不同步
**A:** 打开 Mac 系统设置 → Apple ID → iCloud → 确认 iCloud Drive 已开启

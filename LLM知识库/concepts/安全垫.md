# 💡 安全垫

## 定义

安全垫是指**现价到止损价的距离百分比**，用于衡量"如果看错了，还能亏多少"。安全垫越大，容错空间越大。

## 计算公式

```
安全垫 = (现价 - 止损价) / 现价 × 100%
```

## 一句话解释

> 安全垫 = "最多还能亏多少"，安全垫12%意味着最多亏损12%。

## 实战标准

| 安全垫 | 评级 | 说明 |
|--------|------|------|
| >10% | 🟢 优秀 | 容错空间充足 |
| 7-10% | 🟡 合格 | 可接受，接近临界 |
| <7% | 🔴 危险 | 容错空间不足，慎入 |

## 在天策选股中的应用

- 安全垫 >7% 是基本门槛
- 理想状态：安全垫 ≥10% + SSL金叉已确认
- 安全垫 <7% 即使评分高，也需要严格控仓

## 与仓位的配合

震荡市（regime_pass=false）下：
- 安全垫 >10%：单只仓位可到33%
- 安全垫 7-10%：单只仓位建议≤20%
- 安全垫 <7%：单只仓位≤10% 或不参与

## 典型案例（2026-04-20）

- 通达股份（002560）：安全垫 12.7% → 🟢 优秀
- 康恩贝（600572）：安全垫 7.2% → 🟡 合格
- 汉缆股份（002498）：安全垫 12.2% → 🟢 优秀

## 常见误区

- ❌ 以为安全垫高就可以重仓 → 安全垫高≠一定涨
- ❌ 忽视安全垫薄的风险 → 容错空间不足时被震出

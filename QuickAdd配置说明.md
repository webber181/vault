# QuickAdd 配置说明

> 目标：把 TG 财经资讯、日记、文章、概念、项目的录入动作做成低摩擦快捷入口。

## 1. 菜单总览

建议 QuickAdd 菜单固定为：

- `日记`
- `文章`
- `概念`
- `项目`
- `财经日报`
- `资讯卡`
- `决策记录`
- `板块专题`

## 2. 对应关系

| 命令 | 类型 | 模板 | 保存位置 |
|---|---|---|---|
| 日记 | Macro | `daily-note.md` | `100-日记/` |
| 文章 | Capture / Macro | `article-note.md` | `300-文章/` |
| 概念 | Capture / Macro | `concept-note.md` | `200-概念/` |
| 项目 | Capture / Macro | `project-note.md` | `400-项目/` |
| 财经日报 | Macro | `tg-finance-daily.md` | `投资/` |
| 资讯卡 | Capture / Macro | `tg-finance-item.md` | `投资/` |
| 决策记录 | Capture / Macro | `tg-finance-decision.md` | `投资/` |
| 板块专题 | Capture / Macro | `tg-theme-note.md` | `投资/` |

## 3. 推荐配置原则

### 3.1 日记
- 只问一个信息：今天要写什么
- 自动打开当天日记
- 光标放到“今日计划”或“随手记”区域

### 3.2 文章
- 问：标题、来源、链接
- 自动套 `article-note.md`
- 保存到 `300-文章/`

### 3.3 概念
- 问：概念名
- 自动套 `concept-note.md`
- 保存到 `200-概念/`

### 3.4 项目
- 问：项目名
- 自动套 `project-note.md`
- 保存到 `400-项目/`

### 3.5 财经日报
- 用于当天 TG 整点资讯汇总
- 只写精华，不贴原文
- 保存到 `投资/` 或你指定的日报目录

### 3.6 资讯卡
- 用于单条重大资讯
- 固定保留：时间、来源、原文摘要、核心结论、影响判断、交易含义

### 3.7 决策记录
- 用于资讯触发的交易动作
- 固定保留：触发资讯、核心判断、影响范围、动作、风险、复盘点

### 3.8 板块专题
- 用于 AI / 银行 / 证券 / 半导体 / 医药 等长期跟踪
- 固定保留：板块定义、主线、催化、核心逻辑、关键观察指标

## 4. 实际录入流程

### 4.1 TG 财经整点消息
1. 看到群消息
2. 判断是否为 S / A 级
3. 如果是单条关键消息，用 `资讯卡`
4. 如果是当日汇总，用 `财经日报`
5. 如果会影响操作，用 `决策记录`
6. 如果属于长期主线，补到 `板块专题`

### 4.2 盘前准备联动
- 资讯影响次日开盘判断时，直接链接到 `投资/盘前准备/`
- 资讯影响市场状态和候选股时，链接到 `投资/天策系统/`

## 5. 推荐快捷键

如果你愿意按快捷键记忆，建议这样设：

- `Ctrl+1`：日记
- `Ctrl+2`：文章
- `Ctrl+3`：概念
- `Ctrl+4`：项目
- `Ctrl+5`：财经日报
- `Ctrl+6`：资讯卡
- `Ctrl+7`：决策记录
- `Ctrl+8`：板块专题

### 决策记录

- 当前主线
- 近期催化
- 风险点
- 交易逻辑

## 7. 结论

> QuickAdd 的价值不是把功能做多，而是把你最常用的录入动作压缩到最少点击。
>
> 对 TG 财经资讯，最重要的是：**快速分流、快速提炼、快速归档**。
